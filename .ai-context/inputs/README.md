# Inputs Folder

Drop raw input files here before your AI session:

- Call transcripts from Teams/Zoom → `.vtt` files
- Meeting notes → `.md` or `.txt` files  
- Client emails (copy text) → `.md` files
- Any other raw source material

## Naming Convention
`[YYYY-MM-DD]-[description].[ext]`

Examples:
- `2024-02-20-kickoff-call.vtt`
- `2024-02-21-client-email-requirements.md`
- `2024-02-22-workshop-notes.txt`

## After Processing
Once AI has processed an input file into `outputs/requirements.md` or another output,
you can archive or delete the input. Keep it if you may need to re-process.
